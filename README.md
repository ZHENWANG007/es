# es
人力资源管理系统
一、项目简介
本项目是一套基于前后端分离架构开发的人力资源管理系统。系统涵盖员工、考勤、部门、岗位等核心人事管理功能，并提供多角色权限控制与数据可视化能力，界面设计美观、功能模块完善、代码架构规范，可直接用于小型企业人事管理场景。
二、技术栈
1.后端技术
核心语言：Java
开发框架：SpringBoot
构建工具：Maven
数据库：MySQL（9 张核心业务表）
缓存：Redis（提升系统性能）
2.前端技术
核心框架：Vue
三、系统特点
1.多角色权限控制：支持管理员、HR、普通用户等多种角色，权限粒度精细
2.数据可视化：核心业务数据可视化展示，直观呈现人事 / 考勤等关键指标
3.前后端分离：解耦前后端开发，便于维护和扩展
4.界面体验优秀：页面设计美观，交互逻辑清晰，操作便捷
5.功能全面：覆盖人力资源管理核心场景，满足企业基础人事管理需求
四、核心功能
1.基础功能：登录注册、多角色身份验证
2.数据统计：核心人事 / 考勤数据可视化统计
3.组织架构管理：部门管理、岗位管理、调岗管理
4.考勤管理：我的考勤、考勤记录、考勤统计
5.假期管理：请假记录、请假审批流程
6.系统管理:角色管理、菜单管理
7.通用能力:数据列表分页、条件筛选
